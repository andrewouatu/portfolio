List To Table

GitHub

List To Table is a dual interface application designed to help people find a place to eat and reserve a table as easily as possible.
Customers can quickly check how busy a restaurant is and remotely add themselves to a waitlist. Restaurants can efficiently manage their waitlist and notify customers as soon as a table is available.

  • Worked with a team of 5 developers to build and complete this project.
  • My responsibilities included creating various React components, integrating the Google Places Autocomplete API, and adding styling with the usage of Materialize and CSS3.
  • Technologies: React.js, Redux, JavaScript, HTML5, CSS3, Materialize, PHP, Node.js, MySQL, Google Places API, Google Maps Distance Matrix API, Google Places Autocomplete API, Twillio API
